KuCoin API auto-sync

Community members can optionally link their own KuCoin read-only API key when submitting a bot. The dashboard will automatically re-fetch their stats every 5 minutes using the KuCoin GET /api/v2/grid/spot/bots endpoint, removing the need for manual paste updates.

* Please note: KuCoin does not yet publish a REST API for Grid Spot bots. This feature is currently disabled and will be turned on as soon as KuCoin officially supports it in their API.

Only read-only keys are ever stored — no trading permissions required or accepted.


What is Buffer vs BE?

This computed metric tells you how far the live EQTY price can drop before your bot starts losing money.

• A +5.00% buffer means you can sustain a 5% market drop and still be in profit.
• A -2.00% buffer means your bot is underwater, and the token needs to pump 2% just for you to break even.

What auto-sync fetches

  • Current investment & total profit
  • Grid profit vs unrealized PNL
  • Arbitrage count (24h & total)
  • Bot status (running / stopped)
  • Break-even & APR

API keys are stored encrypted in Supabase and are never exposed to other users. You control deletion from your settings.

Generate a read-only API key in KuCoin → Settings → API Management. Never share a key that has trading or withdrawal permissions.
